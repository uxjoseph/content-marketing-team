# 비주얼 콘텐츠 가이드

인스타그램 카드뉴스와 유튜브 썸네일 가이드입니다.

## 카드뉴스 스펙

### 기본 정보
- **슬라이드 수**: 5-7장
- **크기**: 1080x1350px (인스타 세로형)
- **비율**: 4:5
- **출력**: `outputs/visuals/card-news/slide-01.png ~ slide-07.png`

### 슬라이드 구조

1. **슬라이드 1 (표지)**: 제목 + 후킹 문구
2. **슬라이드 2-6 (본문)**: 핵심 포인트 (1슬라이드 = 1포인트)
3. **마지막 슬라이드 (CTA)**: 저장/팔로우 유도

### 디자인 원칙
- 큰 텍스트 (모바일에서 가독성)
- 단순한 배경
- 일관된 색상 팔레트
- 아이콘/일러스트 활용

---

## 썸네일 스펙

### 기본 정보
- **해상도**: 1280x720px
- **비율**: 16:9
- **출력**: `outputs/visuals/thumbnail.png`

### 구성 요소
- 제목 텍스트 (3-5단어)
- 강조 요소 (화살표, 동그라미 등)
- 인물/제품 이미지 (해당시)
- 브랜드 로고 (선택)

### 디자인 원칙
- 높은 대비 (눈에 띄게)
- 큰 텍스트 (모바일에서도 보이게)
- 감정 유발 색상
- 깔끔한 구도

---

## 이미지 생성 방법

### 옵션 1: Antigravity generate_image 도구
Antigravity의 `generate_image` 도구로 직접 생성:

```
프롬프트 예시:
"인스타그램 카드뉴스 슬라이드, 제목: [제목], 스타일: 미니멀, 색상: 파란색 그라데이션"
```

### 옵션 2: 나노바나나 API (외부)
나노바나나 3.0 Pro API 사용:

```python
# API 호출 예시 (플레이스홀더)
API_KEY = "YOUR_NANOBANANA_API_KEY"
API_URL = "https://api.nanobanana.com/v3/generate"

response = requests.post(
    API_URL,
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={
        "prompt": prompt,
        "size": "1080x1350",  # 카드뉴스
        "style": "professional"
    }
)
```

## 프롬프트 생성 가이드

### 카드뉴스 프롬프트 구조

```markdown
# 카드뉴스 프롬프트

## 전체 테마
- 주제: [브리프의 메인 주제]
- 색상 톤: [권장 색상]
- 스타일: 미니멀/모던/전문적

## 슬라이드별 프롬프트

### 슬라이드 1 (표지)
- 텍스트: "[제목]"
- 서브텍스트: "[후킹 문구]"
- 배경: [배경 설명]

### 슬라이드 2-6 (본문)
- 텍스트: "[핵심 포인트 N]"
- 아이콘: [관련 아이콘 제안]

### 마지막 슬라이드 (CTA)
- 텍스트: "저장하고 나중에 써먹으세요!"
- 아이콘: 저장/북마크 아이콘
```

### 썸네일 프롬프트 구조

```markdown
# 썸네일 프롬프트

- 메인 텍스트: "[3-5단어 제목]"
- 강조 색상: [눈에 띄는 색상]
- 감정: [호기심/놀람/기대]
- 배경: [배경 설명]
- 추가 요소: [화살표/동그라미/인물 등]
```
