# X 스레드 작성 가이드

X(트위터) 바이럴 스레드 10개 트윗 작성 가이드입니다.

## 스펙

- **트윗 개수**: 10개
- **각 트윗 분량**: 280자 이내 (권장 200-250자)
- **구조**: 훅 → 맥락 → 본문 → 사례 → 실전팁 → CTA
- **출력**: `outputs/threads/thread-01.md ~ thread-10.md`

## 스레드 흐름

1. **훅** (호기심 유발)
2. **맥락** (왜 중요한지)
3. **본문** (핵심 내용)
4. **사례/증거** (신뢰성)
5. **실전팁** (적용법)
6. **CTA** (행동 유도)
