# 유튜브 대본 작성 가이드

유튜브 리뷰/해설 영상 대본 작성 가이드입니다.

## 스펙

- **형식**: 타임스탬프 + 화면 구성 포함 스크립트
- **구성**: 인트로(30초) + 본문(섹션당 2분) + 마무리(30초)
- **제목**: 50자 이내, SEO 최적화
- **출력**: `outputs/youtube-script.md`

## 구조

```markdown
# 유튜브 영상 대본

## 영상 정보
- **제목**: [SEO 최적화 제목 - 50자 이내]
- **썸네일 텍스트**: [3-5단어]
- **예상 길이**: [분:초]
- **카테고리**: [교육/리뷰/브이로그 등]

---

## 타임스탬프
- 0:00 인트로
- 0:30 [첫 번째 주제]
- 2:00 [두 번째 주제]
- 4:00 [세 번째 주제]
- 6:00 [네 번째 주제]
- 8:00 마무리

---

## 스크립트

### [0:00-0:30] 인트로 (HOOK)

**[화면: OOO 보여주기]**

[호스트]
"[강력한 오프닝 - 영상을 끝까지 보게 만드는 약속]"

"오늘 영상에서는 [주제]에 대해 알아볼 건데요,
끝까지 보시면 [시청자가 얻을 것]을 알 수 있어요."

**[화면: 채널 로고 / 인트로 영상]**

---

### [0:30-2:00] 섹션 1: [제목]

**[화면: OOO 보여주기]**

[호스트]
"먼저 [첫 번째 주제]부터 이야기할게요."

"[설명 내용]"

**[화면: 그래픽/자료화면 - OOO]**

> 💡 **화면 노트**: [편집자를 위한 지시사항]

---

### [2:00-4:00] 섹션 2: [제목]

**[화면: OOO]**

[호스트]
"두 번째로 알아볼 건 [두 번째 주제]예요."

---

(섹션 3, 4 계속...)

---

### [8:00-8:30] 마무리 + CTA

**[화면: 요약 그래픽]**

[호스트]
"오늘 내용 정리하면요,
첫째, [요약 1]
둘째, [요약 2]
셋째, [요약 3]"

**[화면: 좋아요/구독 애니메이션]**

"이 영상이 도움이 됐다면 좋아요와 구독 부탁드려요.
궁금한 점 있으면 댓글로 남겨주세요."

**[화면: 엔드스크린 - 추천 영상]**

---

## 화면 구성 체크리스트
- [ ] B-roll 필요 장면 표시
- [ ] 그래픽/자료화면 목록
- [ ] 자막 강조 부분 표시
- [ ] 효과음/BGM 포인트

## SEO 정보
- **메인 키워드**: [키워드]
- **태그**: [태그1], [태그2], [태그3]
- **설명란 첫 줄**: [검색 최적화 문장]
```

## 훅(Hook) 유형

### 질문형
- "혹시 OOO 하신 적 있으세요?"
- "왜 OOO는 OOO 할까요?"

### 약속형
- "이 영상 끝까지 보시면 OOO 하실 수 있어요"
- "오늘 알려드릴 3가지만 알면 OOO 됩니다"

### 충격형
- "OOO, 사실 다 거짓말이었어요"
- "아무도 안 알려주는 OOO의 진실"

### 스토리형
- "제가 처음 OOO 했을 때..."
- "3년 전, 저는 OOO 상태였어요"

## 화법

- 자연스러운 구어체
- 한 문장 짧게 (읽기 쉽게)
- 숨 쉴 곳 표시 ("/")
- **[화면: OOO]** 형식으로 시각 요소 표시

## 품질 체크리스트

- [ ] 타임스탬프가 논리적인가?
- [ ] 훅이 30초 안에 시청자를 잡는가?
- [ ] 화면 구성 지시가 명확한가?
- [ ] CTA가 포함되었는가?
- [ ] `guides/style-guide.md` 기준 준수
